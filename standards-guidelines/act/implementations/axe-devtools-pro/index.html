Axe DevTools Pro ACT Implementation

This page is under development and has not been approved by the working group.

Implementation Summary

Axe DevTools Pro version 4.24.5 is an Semi-automated tool by Deque Systems. This ACT implementation tests for WCAG 2.1 Level A & AA. See understanding ACT consistency.

About Axe DevTools Pro Results

Results in this page are taken from a public test report published by Deque Systems. Data is published using the EARL+JSON-LD data format.

Implementation Details

HTML element language subtag matches language

This rule is implemented by Axe DevTools Pro using the lang-change-not-valid (structure) procedure. The implementation is partially consistent with the HTML element language subtag matches language proposed rule. It covers 11 of the 14 examples. For 3 examples no results were reported. This can happen when examples are newer then the test results. The implementation correctly reports which success criteria are failed by this rule. See understanding ACT consistency.
